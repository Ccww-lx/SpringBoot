SpringBoot运行原理：    
+ 入口类和@SpringBootApplication  
+ Starter pom  自定义Strater

核心注解：
+ 注解解析：比如：@ConditionOnBean、 @ConditionOnClass等
+ 自定义

基础配置：
+ banner配置
+ Spring Boot配置文件（properties、yaml）
+ xml文件配置

外部配置：
+ 命令行参数配置
+ 常规属性配置、类型安全的配置

日志配置：
+ 支持Log4J、Log4J2以及Logback（默认）

Profile配置：
+ 不同环境对应不同配置：application-{profile}.properties
